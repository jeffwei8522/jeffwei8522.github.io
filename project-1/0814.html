貳、營運計劃

路線簡介

P.10-12

整個通學車的路線分為兩條：西區線以及東區線，每條路線配置一輛甲類大客車，上下午各行駛一趟次。

西區線 (竹北市公所 <-> 各高中)

早上 (06:35發車): 竹北市公所 → 家樂福 → 縣體育館 → 喜來登 → 水源地(竹商) → 新竹高中 → 建功高中。

下午 (17:15發車): 新竹高中 → 水源地 → 建功高中 → 喜來登 → 縣體育館 → 家樂福 → 竹北市公所。

東區線 (高鐵新竹站 <-> 各高中)

早上 (06:45發車): 高鐵新竹站 → 水圳公園 → 新瓦屋 → 水源地 → 新竹高中 → 建功高中。

下午 (17:15發車): 新竹高中 → 水源地 → 建功高中 → 新瓦屋 → 水圳公園 → 高鐵站。

營運車輛與安全監控

P.13-14
  • 車輛：預定以出廠12年的成運甲類大客車遊覽車行駛。
  • 安全監控系統：

    • 備有公車動態資訊管理系統與瞰車大，以應對未來轉變為公車的需求。
